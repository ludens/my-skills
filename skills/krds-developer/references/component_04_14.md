# page title

파비콘은 브라우저 탭, 즐겨찾기, 방문 기록, 모바일 홈 화면 등에서 서비스의 출처와 정체성을 한눈에 식별할 수 있도록 제공하는 작은 아이콘을 말하며, 공공 웹·모바일 서비스의 신뢰성과 일관성을 높이기 위한 필수 시각 요소이다.

* 개요
* 코드

이 페이지의 구성

### 유형

#### 표준형

(대상 : 정부 상징 로고를 사용하는 중앙행정기관(부/처/청), 특별지방행정기관, 부속기관 등)

#### 확장형

(대상 : 독자적인 로고를 사용하는 중앙기관 (운영 서비스·시스템) 및 공공기관 (대표·운영서비스·시스템)

### 사용성 가이드라인

정부기관의 식별성을 높이기 위해 기관 CI 기반 파비콘을 필수로 제공한다.

브라우저 탭·즐겨찾기·모바일 홈 화면에서 기관/서비스를 빠르게 식별할 수 있도록 기관 CI 기반 파비콘을 필수로 제공한다. 파비콘은 16px에서도 인지 가능하도록 복잡한 디테일을 제거하고 단순화된 심볼 또는 약칭으로 제작하며, 배경과의 명도·색상 대비를 충분히 확보해 어떤 환경에서도 또렷하게 보이도록 한다. 대표 도메인과 서브 서비스에 동일 파비콘을 적용해 이용자가 동일 기관 서비스임을 인지할 수 있게 한다.

모범 사례

라이트·다크 테마 모두에서 구분되도록 버전을 마련한다.

라이트·다크 테마 및 브라우저 및 OS UI 배경 변화에도 식별성이 유지되도록 테마 대응 버전을 제공하며 동일한 심볼 형태를 유지한 채 명도 반전, 외곽선, 배경판 적용 등으로 충분한 대비를 확보한다.
